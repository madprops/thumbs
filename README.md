## Automatic Install

Install `pipx` first.

Then do `pipx install git+https://github.com/madprops/thumbs --force`

Now you have `thumbs` available as a program.

## Manual Install

Go to your code directory.

Git clone this: `git clone https://github.com/madprops/thumbs`

Enter the directory: `cd thumbs`.

Run `./install.sh`

This installs the dependencies.

Then use `./run.sh` to run it.

For instance `/home/me/code/thumbs/run.sh /some/video.mp4`.

To update the code just run `git pull` while inside the directory.
